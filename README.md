**[Canary](https://github.com/vikashftw/Canary)** · a dual-oracle QA swarm that catches the bugs that don't throw. A browser agent declares what it expects, Sentry reports what actually broke, and the two get matched by trace ID and turned into regression tests. Built with two friends at the UC Berkeley AI Hackathon.
